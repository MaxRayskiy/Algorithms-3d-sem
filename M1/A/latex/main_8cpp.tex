\doxysubsection{Detailed Description}
Найдите все вхождения шаблона в строку. Длина шаблона – p, длина строки – n. p $<$= 30000, n $<$= 300000.

Использовать один из методов\+:
\begin{DoxyItemize}
\item С помощью z-\/функции.
\end{DoxyItemize}

Время\+: O(n + p), Доп. память\+: O(p).
